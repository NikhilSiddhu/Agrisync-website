System Architecture

From soil to screen —
every layer.

A complete end-to-end data pipeline spanning sensor nodes in the field, wireless LoRa transmission, edge computing on Raspberry Pi, cloud infrastructure, and a real-time web dashboard.

01
Field Sensor Nodes  ×2–3
Layer 01

Each node sits in the soil and continuously reads key agronomic parameters. A microcontroller onboard processes and packages the readings before wireless transmission.

Soil MoistureSoil TemperatureSoil pHNPK LevelsMicrocontroller
02
Environmental Sensor Node
Layer 02

A dedicated ambient node captures external conditions — temperature, humidity, light, and atmospheric pressure — giving the AI a full picture of the local environment.

Air TemperatureHumidityBarometric PressureLight Intensity
03
LoRa Wireless Transmission
Layer 03

All nodes communicate via LoRa — a long-range, low-power radio protocol providing kilometre-scale reach with zero cellular or Wi-Fi infrastructure. Data hops from field to home entirely wirelessly.

LoRa RadioLow PowerLong RangeMesh Capable
04
Raspberry Pi Gateway
Layer 04

A Raspberry Pi at home acts as the LoRa gateway and edge processor. It receives packets, decodes them, and pushes clean structured data upstream to the cloud. Coded entirely on Raspberry Pi and iPad.

Raspberry PiPythonMQTTEdge Processing
05
Cloud Infrastructure
Layer 05

Data lands in cloud storage where it is indexed and made available for real-time queries. IoT Core ingests the stream, a database stores the historical record, and serverless functions handle processing and AI triggers.

IoT CoreDatabaseServerless FunctionsObject StorageAPI Gateway
06
AI Prediction Engine
Layer 06

A machine learning model trained on simulated historical field data checks current sensor readings against known risk patterns — generating a 12-hour lookahead forecast with disease alerts and actionable recommendations.

Synthetic Training Data12-Hour ForecastDisease DetectionFertilizer RecsPest Alerts
07
agrisync.tech Dashboard
Layer 07

The public-facing web platform presents live field data, AI alerts, historical trends, and recommendations in a clean, accessible interface — built for farmers, agronomists, and researchers alike.

Live DashboardAlert SystemData VisualisationMobile Friendly
