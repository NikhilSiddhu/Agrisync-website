System Architecture

From soil
to screen — every layer.

A complete end-to-end data pipeline spanning sensor nodes in the field, wireless LoRa transmission, edge computing on Raspberry Pi, cloud infrastructure on Cloud, and a real-time web dashboard.

```
🌱 Field Sensor Nodes (×2–3)
Layer 01

Each node sits in the soil and continuously reads key agronomic parameters. A microcontroller onboard processes and packages the readings before wireless transmission.

Soil Moisture Soil Temperature Soil pH NPK Levels Microcontroller
🌤️ Environmental Sensor Node
Layer 02

A dedicated ambient node captures external conditions from the field area — temperature, humidity, light, and atmospheric pressure — giving the AI a full picture of the local environment.

Air Temperature Humidity Barometric Pressure Light Intensity
📡 LoRa Wireless Transmission
Layer 03

All nodes communicate via LoRa (Long Range radio), which provides kilometer-scale range on minimal power — perfect for remote fields with no cellular or Wi-Fi coverage. Data hops from the field to home with zero infrastructure.

LoRa 825-853MHz Low Power Long Range Mesh Capable
🖥️ Raspberry Pi Gateway
Layer 04

A Raspberry Pi at home acts as the LoRa gateway and edge processor. It receives packets, decodes them, and pushes cleaned, structured data upstream to Cloud. Coded entirely on Raspberry Pi and iPad.

Raspberry Pi Python MQTT / HTTP Edge Processing
☁️ Cloud Infrastructure
Layer 05

Data lands in Cloud servie where it is stored, indexed, and made available for real-time queries. IoT Core ingests the stream, DynamoDB or S3 stores the historical record, and Lambda functions handle processing and AI triggers.

Cloud IoT Core DynamoDB Lambda S3 API Gateway
🤖 AI Prediction Engine
Layer 06

A machine learning model trained on simulated historical field data checks current sensor readings against known risk patterns. It generates a 12-hour lookahead forecast — alerting farmers to potential disease outbreaks, pest conditions, or nutrient stress before they occur, with actionable recommendations.

Synthetic Training Data 12-Hour Forecast Disease Detection Fertilizer Recs Pest Alerts
🌐 agrisync.tech Dashboard
Layer 07

The public-facing web platform at agrisync.tech presents live field data, AI alerts, historical trends, and recommendations in a clean, accessible interface — built for farmers, agronomists, and researchers.

Live Dashboard Alert System Data Viz Mobile Friendly
```
